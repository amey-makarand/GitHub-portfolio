Flood Damage Dection using Satellite Images

Developed Convolutional Neural Network (CNN) models using PyTorch, alongside a custom neural network model, to detect hurricane-induced damage utilizing NOAA’s Hurricane Harvey dataset. The ResNet50 model achieved 93.56% accuracy, while GoogleNet reached 90.65% on a custom dataset. The custom model demonstrated comparable performance, validating its effectiveness.s Published a paper on these findings at the AIR 2022 conference, hosted by Motilal Nehru National Institute of Technology Allahabad. As the corresponding author, I led the submission, peer review, and publication processes. This was part of my final year project.
Key features of the project include:

• Comparsion betweeen the 12 prebuilt models from Pytorch and the custom model.
• Creating a customised dataset and finding comparable results wrt to the first dataset.

bc-command

Implemented the bc command in C. The bc command in Linux stands for “basic calculator” and is a powerful command-line utility that allows users to perform arbitrary-precision arithmetic operations. The project was implemented using linked list and the stack data structure. It is a terminal based project. This was part of my Data Structures and Algorithms Course
Key features of the project include:

• Arithmetic Operations: Supports addition, subtraction, multiplication, division, and modulus operations.
• Advanced Functions: Includes functions like bitwise shift operations.
